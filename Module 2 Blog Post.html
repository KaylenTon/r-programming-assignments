Module # 2 Assignment Importing Data and Function Evaluation in R

September 8, 2025

Assignment Code

assignment2 <- c(16, 18, 14, 22, 27, 17, 19, 17, 17, 22, 20, 22)
myMean <- function(assignment2) {
  return(sum(assignment) / length(someData))
}
myMean(assignment2)

The Error Message

>   return(sum(assignment2) / length(someData))
Error: object 'someData' not found

The Reasons

In the function-creating line of code, it uses variables such as ‘assignment’ and ‘someData’. These aren’t existing variables/objects. The function intends to calculate the mean of the argument, which in this case is ‘assignment2’. Because it is so specific to assignment number two and not one, three, and etc., we need to use data from the assignment2 vector. Thus, replacing the variables ‘assignment’ and ‘someData’ with ‘assignment2’ should fix this error.

Corrected Code

myMean <- function(assignment2) {
  return(sum(assignment2) / length(assignment2))
}

And when I run: myMean(assignment2), this is displayed:

> myMean(assignment2)
[1] 19.25

That is correct!
